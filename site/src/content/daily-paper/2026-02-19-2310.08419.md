# Jailbreaking Black Box Large Language Models in Twenty Queries

Most adversarial attack research assumes white-box access to model weights and gradients. But deployed systems are black-box: you see only the text output. The practical question is: how easy is it to jailbreak a system through API queries alone, without any access to internals?

PAIR (Prompt Automatic Iterative Refinement) answers this with uncomfortable efficiency. An attacker LLM generates candidate jailbreak prompts, observes which ones work best, and refines them iteratively. Within 20 API queries, the system can reliably find prompts that cause GPT-4, Claude, or Gemini to violate their safety guidelines. The attack works because refusal patterns are predictable—when a model refuses, it typically explains why (e.g., "I can't help with that because..."), and the attacker model learns from these explanations to generate improved attacks. This creates a feedback loop where refusals become training data for generating better attacks.

The implications are stark. Black-box jailbreaking is cheap (fractions of a cent in API costs), fast (minutes of work), and effective. This means organizations deploying models through APIs cannot rely on the assumption that "it's hard to jailbreak because you don't have source access." In fact, the barrier to attack has shifted from "requires ML expertise" to "requires an API key and another LLM." For defenders, this means assuming attackers will use automated, iterative discovery rather than manual prompt engineering.

## Abstract

There is growing interest in ensuring that large language models (LLMs) align with human values. However, the alignment of such models is vulnerable to adversarial jailbreaks, which coax LLMs into overriding their safety guardrails. The identification of these vulnerabilities is therefore instrumental in understanding inherent weaknesses and preventing future misuse. To this end, we propose Prompt Automatic Iterative Refinement (PAIR), an algorithm that generates semantic jailbreaks with only black-box access to an LLM. PAIR -- which is inspired by social engineering attacks -- uses an attacker LLM to automatically generate jailbreaks for a separate targeted LLM without human intervention. In this way, the attacker LLM iteratively queries the target LLM to update and refine a candidate jailbreak. Empirically, PAIR often requires fewer than twenty queries to produce a jailbreak, which is orders of magnitude more efficient than existing algorithms. PAIR also achieves competitive jailbreaking success rates and transferability on open and closed-source LLMs, including GPT-3.5/4, Vicuna, and Gemini.
